Student Leave Approval Automation using n8n

Overview

This project automates the student leave approval process using n8n, Google Forms, Google Sheets, and Gmail.

Features

- Collects leave requests through Google Forms
- Stores responses in Google Sheets
- Automatically evaluates leave requests
- Approves leave requests up to 3 days
- Declines leave requests exceeding 3 days
- Sends email notifications to students
- Updates leave status automatically

Tools Used

- n8n
- Google Forms
- Google Sheets
- Gmail

Learning Outcomes

- Workflow Automation
- No-Code Development
- Google Workspace Integration
- Conditional Logic in n8n
- Email Automation
